This project uses [mkslides](github.com/MartenBE/mkslides) to render markdown files into [reveal.js](https://revealjs.com/) slideshows.

To get started:

- Install `uv`

```shell
uv run mkslides serve src/
```

Navigate to http://localhost:8000

- Create a new markdown file in `src/`
- Markdown examples can be found here: https://github.com/HoGentTIN/hogent-markdown-slide
- Also reference reveal.js docs
